# Kreta 2026 🇬🇷

## Slik deployer du på GitHub Pages

1. Last opp `index.html` til dette repoet
2. Gå til **Settings** → **Pages** (i venstremenyen)
3. Under **Source**: velg **Deploy from a branch**
4. Under **Branch**: velg `main` og mappen `/ (root)` → klikk **Save**
5. Vent 1–2 minutter, så dukker lenken opp øverst på Pages-siden

## Lokalt

Bare åpne `index.html` i en nettleser — ingen server nødvendig.
